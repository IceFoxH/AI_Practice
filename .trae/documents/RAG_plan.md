# RAG 知识库问答系统搭建计划

## 项目概述
为用户搭建一个基于检索增强生成（RAG）的知识库问答系统，支持文档上传、向量存储、智能问答等功能。

## 技术栈
- **后端框架**: FastAPI + Python
- **RAG框架**: LangChain
- **向量数据库**: FAISS (本地轻量级)
- **LLM模型**: OpenAI API (支持配置其他模型)
- **前端**: HTML/CSS/JavaScript (简洁界面)

## 系统架构
```
┌─────────────────────────────────────────────────────┐
│                   前端界面                          │
├─────────────────────────────────────────────────────┤
│              FastAPI 后端服务                        │
│  ┌──────────┐  ┌──────────┐  ┌──────────────────┐  │
│  │文档加载器│→│文本分割器│→│向量数据库(FAISS)│  │
│  └──────────┘  └──────────┘  └────────┬─────────┘  │
│                                       │             │
│  ┌────────────────────────────────────┼─────────┐   │
│  │           问答引擎                 ↓         │   │
│  │  检索器 → 上下文构建 → LLM生成答案          │   │
│  └─────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

## 文件结构
```
rag-system/
├── app.py                 # FastAPI 主应用
├── rag_engine.py          # RAG核心逻辑
├── config.py              # 配置文件
├── requirements.txt       # 依赖列表
├── documents/             # 知识库文档目录
│   └── sample.txt         # 示例文档
└── static/                # 静态文件
    └── index.html         # 前端界面
```

## 功能模块
1. **文档管理模块** - 上传、加载文档
2. **向量存储模块** - 将文档转换为向量并存储
3. **检索模块** - 根据问题检索相关文档
4. **问答模块** - 结合上下文生成答案
5. **API接口** - RESTful API服务
